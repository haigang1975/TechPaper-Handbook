本项目旨在为科研人员、工程师及学术写作者提供一份详尽、规范的科技论文写作指导手册。内容涵盖了从选题、提纲拟定、数据展示到特定领域（建筑、信息技术）写作技巧的全方位指南，帮助作者提升论文质量与发表成功率。

## 📖 项目简介

这份文档整理自资深编辑与学术写作专家的经验总结（Version 0.3, 2025-01）。它不仅包含了通用的科技论文写作规范（如摘要撰写、图表规范、参考文献引用），还特别针对**建筑类**和**信息技术类**论文提供了专业的领域特定指导。

无论你是准备投稿期刊，还是撰写毕业论文，这份指南都能为你提供实用的参考。

## 📑 目录结构

本指南主要分为三个核心部分：

### 第一部分：科技论文写作基础
*   **准备工作**：提纲拟定、题目选取、材料整理。
*   **核心规范**：摘要写作技巧（总分型、总结型等）、引言与结论撰写、文献综述方法。
*   **格式细节**：数学公式排版、表格（三线表）与图片的使用规范、单位与符号的标准化（正斜体、大小写）。
*   **常见问题**：审稿常见拒稿原因分析（主题不明确、创新性不足等）。

### 第二部分：建筑类论文写作基础
*   **选题与类型**：新材料应用、节能施工技术、BIM技术等选题指南。
*   **专业术语**：详细释义了超高性能混凝土(UHPC)、建筑信息模型(BIM)、传热系数等专有名词。
*   **写作要点**：针对试验类论文、景观设计类论文的特定写作逻辑与要求。

### 第三部分：信息技术类论文写作技巧与规范
*   **创新与表达**：如何挖掘算法创新点、理论创新与应用创新的表述。
*   **技术细节**：伪代码与流程图的描述、大数据与AI模型的实验结果展示。
*   **跨学科写作**：信息科学与其他领域的交叉研究写作策略。

## 🛠️ 使用指南

1.  **查阅规范**：在写作前或修改阶段，查阅对应章节（如第8章公式规范、第12章单位使用）以确保格式合规。
2.  **领域参考**：建筑或IT领域的研究者，请重点阅读对应部分，避免领域特定的常识性错误。
3.  **术语核对**：利用文档中的术语表（如第31章）确保专业名词使用准确。
